Data Source : Maharashtra remote sensing agency
Level of data : Village
Data content Overview: 
1. District boundary – area/perimeter, type (hilly block) etc.
2. Taluka boundary, 
3. Village boundary,
4. Drainage – reservoir, river, tank (name of each), area/perimeter of each.
5. Road –highways (state, express etc), districtroads, village roads, length and type of each, taluka of each.
6. Watershed – name of catchment, watershed, basin, area coverd etc.
7. Rail n/w
